Round Timer - Installation Instructions
========================================

Thank you for downloading Round Timer!

QUICK START
-----------

To install Round Timer on your Linux desktop, simply run:

    ./install.sh

This will install the application to your system and add it to your
Applications menu (KDE, GNOME, etc).


WHAT GETS INSTALLED
-------------------

- Binary: ~/.local/bin/round-timer
- Desktop entry: ~/.local/share/applications/round-timer.desktop
- Icon: ~/.local/share/icons/hicolor/scalable/apps/round-timer.svg

No sudo required - everything installs to your home directory.


AFTER INSTALLATION
------------------

You can launch Round Timer:
- From your Applications menu (look under Utilities)
- From a terminal by running: round-timer

If the app doesn't appear in your menu immediately, try logging out
and back in, or restart your desktop environment.


UNINSTALL
---------

To remove Round Timer from your system:

    ./uninstall.sh


REQUIREMENTS
------------

- Linux desktop environment (KDE, GNOME, XFCE, etc)
- ALSA audio libraries (usually pre-installed)
